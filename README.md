# Micro:bit Face Recognition Robot

## Project Overview
This project implements a face recognition system using a micro:bit-based robot, following Agile methodologies.

## Features
- Face detection and recognition
- LED matrix display with number mapping
- Face data protection via SHA-256 hashing
- Agile development with Scrum sprints
- GitHub collaboration

## Hardware Requirements
- Yahboom Tinybit Pro robot kit
- micro:bit V2
- Camera module
- Computer for programming
